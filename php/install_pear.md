# Install PEAR

Steps to install PEAR

## MacOS

**System Wide Setup**

```
curl -O  http://pear.php.net/go-pear.phar

sudo php -d detect_unicode=0 go-pear.phar

```

For a local instance setup, remove the `sudo` from the previous command.
